How this site is built

This website looks simple (on purpose!) but it’s built using the same kind of tools that lots of real companies use.

Code → GitHub → Azure Static Web Apps → You!

Here’s what’s happening:

  • I write the website pages (that’s the “code”).
  • I save the code in GitHub (a safe place for storing and sharing code).
  • GitHub then tells Azure to publish the newest version of the site.
  • Then anyone can open it on a phone, tablet, or computer and start learning.

The fun part is: every time I update a page and press “commit”, the website updates itself!

Who am I?

Hi! I’m the person behind The LEGO Cloud Architect. I’m from (and live in) the UK.

I decided I wanted to “do computers” at school when I was about 14… mostly to impress a girl I was fond of. (A classic IT origin story.)

I studied Advanced IT at college, and got my first IT job when I was 20.

My first IT job (aka: The Engineer Case Era)

My job was to build PCs and travel around to small companies fixing all sorts of things: computers, printers, networks… and the occasional “have you tried turning it off and on again?”

I had a trusty engineer’s case full of tools and cables. It felt a bit like being a tech superhero… just with more paper jams.

How I ended up in the cloud

I first heard about “cloud computing” around 2008. It sounded like magic back then.

By 2012, I was helping set up a small hosting company using private clouds — and I was working from home before it was cool.

Over the years I’ve been an architect and solution builder for service providers, banks, and a large sporting company.

Family life

By 2019 I was married, with my first son (born in 2018). My second son arrived in 2020.

They’re the inspiration for this whole website. They ask big questions — and I love trying to answer them in a way that actually makes sense!

When I’m not doing computers…

I enjoy reading, movies, and of course spending time with my family.
